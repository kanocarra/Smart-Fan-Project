How It Works

Welcome to Fan Base, an interactive real-time dashboard for all your smart fan needs.

This dashboard has been built to showcase how a real-time database can be used to monitor and control the status of a fan inside a Fisher & Paykel B-Model Fridge.

Combined with a node.js server and an Atmel ATTiny 841, Firebase allows us to log data and display it in real-time.

Brought to you as part of the ELECTENG 311 Project
